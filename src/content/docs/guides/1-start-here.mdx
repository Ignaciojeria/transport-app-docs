Interactúe con su sistema de gestión de transporte creando una nueva organización. Al acceder a este endpoint, se generará automáticamente una API Key única que deberá mantener resguardada y que facilitará todas las operaciones del sistema.

## 📨 HTTP Request

```sh
curl --location 'https://api.ngs.global:443/v1/nats/subjects/organization.request' \
--header 'accept: application/json' \
--header 'authorization: nhg_2atLsmWBRcK7z3Mtu_DmLVjm3aHiYcfFQHoLrxWGhLtTZqhoPLfRN3rKmsmSWe' \
--header 'content-type: application/json' \
--data '{
    "name": "nombre-de-tu-organización"
}'
```
